# dApps

## Introduction

This section covers packages and utilities designed to help developers build decentralized applications (dApps) on the Polkadot ecosystem. From key management and cryptographic primitives to account derivation and session handling, these tools provide the building blocks needed to create secure and user-friendly Polkadot-native dApps.

## Packages

- **[Keys](/dapps/keys/)**: Hierarchical key derivation and session key management for Polkadot accounts.
